# ictx

[![npm version](https://badge.fury.io/js/ictx.svg)](https://badge.fury.io/js/ictx)

`ictx` is a new Node.js npm package. Idio context for zoroaster context testing framework.

```sh
yarn add -E ictx
```

const ictx = require('ictx')
